# ? MONDIcare - Irish Potato Disease Detection  
  
AI-powered disease detection for Irish potato farmers.  
  
## Features  
  
- Instant disease measurement (Late Blight, Early Blight, Healthy)  
- Pesticide recommendations  
- Shop locator  
- Connect with agricultural officers  
  
## Model Download  
  
The AI model is downloaded automatically from Google Drive on first run.
